Formulaire

Il s’agit d’un formulaire avec plusieurs champs. Le premier champ est destiné à recueillir le pseudo de l’utilisateur. Le deuxième champ est destiné à recueillir le nom mo de passe de l'utilisateur, avec une étiquette Chaque champ permet à l’utilisateur de saisir du texte.input Les informations saisies peuvent ensuite être récupérées grâce aux infos misent.

Liste

Il s’agit d’une liste non ordonnée contenant trois éléments. Les éléments de la liste sont “Élément 1”, “Élément 2” et “Élément 3”. Dans une liste non ordonnée, les éléments sont généralement affichés avec des puces (•) devant chaque élément. Cette liste pourrait être utilisée pour afficher des informations dans un ordre qui n’a pas d’importance particulière. Par exemple, cela pourrait être une liste de choses à faire, une liste de courses, ou toute autre liste d’éléments où l’ordre n’est pas important en utilisant les balises ul et il
